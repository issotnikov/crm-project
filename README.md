# CRM System — Проектирование системы обработки входящих заявок

> **Статус:** Технический и продуктовый проект  
> **Версия:** 1.0  
> **Дата:** 2026-06-28  
> **Автор:** System Architect (AI-assisted)

## О проекте

Полнофункциональная CRM/ERP-гибрид система для централизованного сбора, обработки и управления входящими заявками и всеми бизнес-процессами компании. Система объединяет 4 канала поступления заявок (Telegram, Email, Телефон, Web-форма) в единую платформу с полным циклом: **Клиент → Заявка → Сделка → Проект → Финансы**.

## Технологический стек

| Слой | Технология | Обоснование |
|------|-----------|-------------|
| Backend | Python (FastAPI) | Async, типизация, скорость разработки |
| Frontend | React + TypeScript | Компонентность, экосистема |
| БД | PostgreSQL 16 | Реляционная, JSONB, полнотекстовый поиск |
| Кэш/Очереди | Redis + Celery | Брокер сообщений, кэширование |
| Поиск | Elasticsearch (опц.) | Полнотекстовый поиск по коммуникациям |
| Файлы | MinIO / S3 | Документы, вложения |
| Деплой | Docker + Kubernetes | Контейнеризация, масштабирование |
