# 07. UI/UX структура системы

---

## 1. Дизайн-система

### 1.1. Цветовая палитра

| Назначение | Цвет | HEX | Применение |
|-----------|------|-----|-----------|
| **Primary** | Индиго | `#4F46E5` | Кнопки CTA, активные элементы, ссылки |
| **Primary Hover** | Индиго тёмный | `#4338CA` | Hover-состояния |
| **Success** | Зелёный | `#10B981` | Успех, оплачено, выполнено |
| **Warning** | Янтарный | `#F59E0B` | Ожидание, предупреждение |
| **Danger** | Красный | `#EF4444` | Ошибка, просрочка, удаление |
| **Info** | Голубой | `#3B82F6` | Информация, уведомления |
| **Background** | Светло-серый | `#F9FAFB` | Основной фон рабочей зоны |
| **Surface** | Белый | `#FFFFFF` | Карточки, таблицы, модальные окна |
| **Text Primary** | Тёмно-серый | `#111827` | Основной текст |
| **Text Secondary** | Серый | `#6B7280` | Подписи, метаданные |
| **Border** | Светло-серый | `#E5E7EB` | Границы, разделители |
| **Sidebar BG** | Тёмно-синий | `#1E1B4B` | Боковое меню (тёмная тема) |

### 1.2. Типографика

| Элемент | Шрифт | Размер | Вес |
|---------|-------|--------|-----|
| H1 (заголовок страницы) | Inter | 28px / 1.75rem | 700 Bold |
| H2 (заголовок секции) | Inter | 22px / 1.375rem | 600 SemiBold |
| H3 | Inter | 18px / 1.125rem | 600 SemiBold |
| Body text | Inter | 14px / 0.875rem | 400 Regular |
| Small / Caption | Inter | 12px / 0.75rem | 400 Regular |
| Table data | Inter | 14px | 400 Regular |
| Numbers / KPI | Inter (tabular) | 32px / 2rem | 700 Bold |

### 1.3. Компонентная библиотека

Основана на **shadcn/ui** (Radix UI primitives + Tailwind CSS).

| Компонент | Назначение |
|-----------|-----------|
| `Button` | Кнопки (primary, secondary, ghost, danger) |
| `Input` / `Textarea` / `Select` | Формы ввода |
| `DataTable` | Таблицы с сортировкой, фильтрами, пагинацией |
| `Modal` / `Dialog` | Модальные окна |
| `Drawer` / `Sheet` | Выезжающие панели (карточка клиента сбоку) |
| `Badge` / `Tag` | Статусы, теги, пилюли |
| `Avatar` | Аватарки пользователей |
| `Tooltip` | Подсказки |
| `Toast` / `Notification` | Всплывающие уведомления |
| `Dropdown` | Выпадающие меню |
| `Tabs` | Вкладки |
| `KanbanBoard` | Канбан-доска (drag-and-drop) |
| `Calendar` | Календарь (FullCalendar) |
| `Timeline` | Лента взаимодействий |
| `Chart` / `Graph` | Графики (Recharts) |

### 1.4. Иконки

- **Библиотека:** Lucide Icons (line, stroke-width 1.5px)
- **Размеры:** 16px (inline), 20px (меню), 24px (заголовки)

### Header (Шапка)

| Элемент | Описание |
|---------|----------|
| Глобальный поиск | Поиск по клиентам, сделкам, задачам (Cmd+K) |
| Кнопка «+ Создать» | Быстрое создание: лид, задача, встреча |
| Колокольчик уведомлений | In-app уведомления (с бейджем) |
| Аватар пользователя | Выпадающее меню: профиль, настройки, выход |

## 7. Адаптивность

| Устройство | Поведение |
|-----------|-----------|
| Desktop (>1280px) | Полный layout: sidebar + header + content |
| Tablet (768–1280px) | Схлопываемый sidebar (только иконки), контент на всю ширину |
| Mobile (<768px) | Sidebar — drawer (выезжает), контент вертикально, канбан — горизонтальный скролл |
| PWA | Offline-кэширование списков, push-уведомления |

### Mobile-особенности

- Bottom navigation bar (Дашборд, Заявки, Задачи, Календарь)
- Swipe-жесты для канбан-доски
- Click-to-call напрямую
- Упрощённые карточки (только ключевая информация)

## 8. Accessibility (a11y)

| Правило | Реализация |
|---------|-----------|
| Контраст текста | WCAG AA минимум (4.5:1 для body text) |
| Keyboard navigation | Tab + Enter + Escape, фокус-стили видимые |
| Screen reader | ARIA labels, semantic HTML, role attributes |
| Font size | Минимум 14px body, масштабируемость до 200% |
| Color independence | Информация не передаётся только цветом (иконки + текст) |
