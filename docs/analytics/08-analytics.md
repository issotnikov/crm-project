## 1. Ключевые метрики (KPI)

### 1.1. Продажи

| Метрика | Формула | Целевое значение | Период |
|---------|---------|------------------|--------|
| **Количество новых лидов** | COUNT(leads WHERE created_in_period) | Рост 10% м/м | День / Неделя / Месяц |
| **Конверсия лида → сделка** | deals_from_leads / total_leads × 100% | 25–40% | Месяц |
| **Конверсия сделка → выиграна** | won_deals / total_deals × 100% | 30–50% | Месяц |
| **Средний размер сделки** | SUM(deal_amount_won) / COUNT(won_deals) | Зависит от ниши | Месяц |
| **Выручка** | SUM(deal_amount_won) | План | Месяц / Квартал |
| **Скорость цикла продаж** | AVG(actual_close_date - created_at) для won deals | Снижение | Месяц |
| **Вероятная выручка (прогноз)** | SUM(amount × probability / 100) для open deals | — | Текущий момент |

### 1.2. Маркетинг (источники лидов)

| Метрика | Формула | Целевое значение |
|---------|---------|------------------|
| **Стоимость лида (CPL)** | campaign_budget / leads_from_campaign | Снижение |
| **Стоимость клиента (CAC)** | campaign_budget / won_deals_from_campaign | < LTV / 3 |
| **ROI канала** | (revenue_from_channel - cost) / cost × 100% | > 200% |
| **LTV (Lifetime Value)** | AVG(total_revenue per customer) | Рост |
| **Доля канала в выручке** | revenue_from_channel / total_revenue × 100% | Баланс |

### 1.3. Эффективность сотрудников

| Метрика | Формула | Целевое значение |
|---------|---------|------------------|
| **Количество обработанных лидов** | COUNT(leads WHERE assigned_to=X AND status≠new) | План |
| **Среднее время реакции (SLA)** | AVG(responded_at - created_at) для leads | < 30 мин |
| **% нарушения SLA** | COUNT(sla_violated) / total_leads × 100% | < 10% |
| **Выручка менеджера** | SUM(won_deals.amount WHERE assigned_to=X) | План |
| **Конверсия менеджера** | won / total_assigned × 100% | Рост |
| **Выполнено задач** | COUNT(tasks WHERE assignee=X AND status=done) | План |
| **% просроченных задач** | overdue_tasks / total_tasks × 100% | < 15% |

### 1.4. Финансы

| Метрика | Формула | Целевое значение |
|---------|---------|------------------|
| **Выставлено счетов** | SUM(invoices.total WHERE issued_in_period) | — |
| **Оплачено** | SUM(payments.amount WHERE paid_in_period) | План |
| **Дебиторская задолженность** | SUM(invoices.total - paid_amount WHERE status≠paid) | Снижение |
| **Просроченная задолженность** | SUM(overdue_invoices.balance) | Минимум |
| **Средний срок оплаты** | AVG(payment_date - issue_date) | Снижение |
| **Доля оплаченных счетов** | paid_invoices / total_sent × 100% | > 80% |

## 5. Графики и визуализации

| Тип графика | Назначение |
|-------------|-----------|
| **Bar chart** | Выручка по месяцам, сравнение периодов |
| **Line chart** | Динамика метрик во времени |
| **Funnel chart** | Воронка продаж |
| **Pie / Donut** | Распределение по источникам, каналам |
| **Stacked bar** | Сделки по стадиям и менеджерам |
| **Heatmap** | Активность по дням/часам |
| **Gauge** | Выполнение плана (KPI) |
| **Gantt** | Проекты и этапы |
